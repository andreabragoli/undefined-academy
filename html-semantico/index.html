HTML Semantico

Por Andrea Bragoli. Published Apr 9th.

Esta es una pagina web de ejemplo que explica el marcado semantico.

El esquema del documento

HTML5 incluye varios elementos de "contenido de seccion que afectan el contorno del documento".

Encabezados

El elemento <header> es uno de esos elementos que "crea secciones".

Pie de paginas

Asi mismo el elemento <footer>.

HTML semantico en linea

El elemento <time> es semantico, pero no crea una seccion de contenido.
